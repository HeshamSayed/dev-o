Quick Start

Install DEVO CLI

curl -fsSL https://devtools-co.com/install.sh | bash

The installer will:

  • Download and install the DEVO CLI
  • Configure connection to the backend
  • Set up your development environment

Getting Started

1. Initialize Project

devo init my-project

Create a new DEVO project with AI agent support

2. Start Chat

devo chat

Begin an interactive session with AI agents

3. Check Status

devo status

View project status and agent activity
